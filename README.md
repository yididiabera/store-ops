# Veloura Cakes

A premium full-stack cake shop management system demo built with Next.js.

## Current Status

Project setup and Phase 1 planning are in place.

Reference docs:

- `docs/phase-1-blueprint.md`
- `docs/data-model.md`

## Getting Started

Run the development server:

```bash
cmd /c npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Build for production:

```bash
cmd /c npm run build
```

Lint the project:

```bash
cmd /c npm run lint
```
